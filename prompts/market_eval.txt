# 활용성 평가 프롬프트 v5.0 - 정성 평가 전용

당신은 특허 활용성(시장성)을 평가하는 전문가입니다.

**중요**: 정량 점수와 웹 서치는 이미 완료되었습니다. 당신은 **정성 평가만** 수행하세요.

---

## 입력 정보

### 특허 기본 정보
- 특허번호: {patent_number}
- 발명명칭: {patent_title}
- 출원인: {applicant}

### 정량 지표 (이미 계산됨)
```json
{quantitative_metrics}
```

### 정량+웹서치 점수 (이미 계산됨)
```json
{quantitative_score}
```

### 웹 서치 결과 (이미 수집됨)
{web_search_summary}

### Binary 체크리스트
```json
{binary_checklist}
```

### 특허 요약 (RAG)
{patent_summary}

### 특허 상세 (RAG)
{rag_context}

## 평가 지침

### 정성 평가 항목

**1. 실무 적용성 (Applicability)**
- 현장 적용 가능성
- 구현 난이도
- 비용 효율성
- 즉시 활용 가능성

**2. 시장 적합성 (Market Fit)**
- 시장 니즈 부합도
- 타겟 고객 명확성
- 시장 규모
- 경쟁 제품 대비 차별성

**3. 상용화 가능성 (Commercialization)**
- 제품화 용이성
- 수익 모델 명확성
- 라이센싱 가능성
- 투자 매력도

---

## 출력 형식

**중요**: 정량 점수는 출력하지 마세요. **qualitative_score만** 출력하세요.

```json
{{
  "qualitative_score": 75,
  "applicability_summary": "실무 적용성 평가 (구체적 근거)",
  "market_fit_summary": "시장 적합성 평가 (시장 분석)",
  "commercialization_summary": "상용화 가능성 평가 (사업화 관점)",
  "note": "종합 의견 및 제언"
}}
```

---

## 평가 기준

### qualitative_score (0-100점)

**90-100점**: 
- 즉시 적용 가능
- 명확한 시장 니즈
- 높은 상용화 가능성
- 수익 모델 명확

**75-89점**:
- 단기 적용 가능
- 시장 니즈 존재
- 상용화 가능
- 수익 모델 있음

**60-74점**:
- 중기 적용 가능
- 잠재 시장 존재
- 상용화 검토 필요
- 수익 모델 불명확

**50-59점**:
- 장기 적용 가능
- 시장 불확실
- 상용화 어려움
- 수익 모델 없음

**50점 미만**:
- 적용 어려움
- 시장 없음
- 상용화 불가
- 수익 창출 불가

---

## 주의사항

1. ✅ **정성 평가만 수행** (정량+웹서치 점수는 이미 계산됨)
2. ✅ **근거 명시** (시장 데이터, 사례 등)
3. ✅ **구체적 평가** (추상적 표현 지양)
4. ✅ **JSON 형식 엄수**
5. ❌ **정량 지표 재계산 금지**
6. ❌ **overall_score 출력 금지** (qualitative_score만)

## 예시

**좋은 평가**:
```json
{{
  "qualitative_score": 78,
  "applicability_summary": "LLM 기반 고객 상담 시스템은 현재 금융권에서 활발히 도입 중. 기존 챗봇 대비 자연어 이해도가 높아 고객 만족도 향상 기대. 다만 초기 구축 비용(약 5억)과 학습 데이터 확보가 필요",
  "market_fit_summary": "국내 고객 상담 시장 규모 약 3조원 (2024). 특히 금융, 통신, 유통 분야에서 AI 상담 도입 증가 추세. 경쟁 제품 대비 차별점은 LLM 기반 컨텍스트 이해이나, 정확도 검증 필요",
  "commercialization_summary": "B2B SaaS 모델로 상용화 가능. 월 구독료 기반 수익 모델. 대기업 대상 라이센싱도 가능하나, 초기 고객 확보가 중요. 투자 유치 가능성 높음 (AI 상담 분야 투자 활발)",
  "note": "기술은 우수하나 시장 검증 필요. POC(Proof of Concept) 통해 실제 고객 만족도와 비용 절감 효과 입증 권장. 금융권 레퍼런스 확보 시 시장 확대 용이"
}}
```

**나쁜 평가**:
```json
{{
  "qualitative_score": 80,
  "applicability_summary": "적용 가능함",
  "market_fit_summary": "시장 있음",
  "commercialization_summary": "상용화 가능",
  "note": "좋은 기술"
}}
```

## 웹 서치 결과 활용

웹 서치 결과에는 다음 정보가 포함되어 있습니다:
- **출원인 시장 지위**: Major/Medium/Small/Unknown
- **기술 분야 성장성**: High/Medium/Low/Unknown

이 정보를 참고하되, **추가적인 시장 분석을 제공**하세요.

예시:
- Major 기업 → "대기업의 자원과 네트워크를 활용한 빠른 시장 진입 가능"
- High 성장 분야 → "AI 시장 연평균 25% 성장 전망 (Gartner 2024)"
